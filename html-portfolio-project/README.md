# HTML Portfolio Project

A beginner-friendly multi-page HTML portfolio created while learning web development fundamentals.

## Overview

This project includes two small HTML projects and supporting About/Contact pages. It focuses on clean HTML structure, links, images, lists, and relative file paths.

## Pages Included

- Home page
- Movie Ranking Project
- Birthday Invite Project
- About Me page
- Contact Me page

## Skills Practiced

- HTML5 boilerplate
- Headings and paragraphs
- Anchor tags and navigation
- Image tags with alt text
- Lists
- Relative file paths
- Multi-page website structure

## Folder Structure

```text
html-portfolio-project/
├── index.html
├── README.md
├── assets/

    ├── contact.html

```

## How to View

Open `index.html` in a browser, or deploy the repository using GitHub Pages.
